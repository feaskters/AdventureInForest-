## 玩法介绍
- 玩家可以通过跳跃和冲刺动作通过关卡，关卡设计中加入了自行动和自动攻击玩家的怪物，以及需要通过一定的操作和思考才能通过的关卡。

## 所学技术
- `cinemachine`相机平缓跟随
- `Light2D`2d游戏光照系统(试验版本)
- `TileMap`绘制地图
- 单例控制全局音乐
- 角色技能创建
- boss行动模式控制
- 伤害判定
- 怪物追随
- 无敌时间判定
- 特效使用
- 第三方摇杆控件使用及适用性修改
- 转场动画
